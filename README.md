# Address Book in C

## Description
Address Book is a simple contact management system developed using C programming. It allows users to store and manage contact details such as name, phone number, and email address. The project uses structures and file handling for efficient data management.

## Features
- Add new contacts
- View all contacts
- Search contacts
- Edit contact details
- Delete contacts
- Save contacts permanently using file handling

## Technologies Used
- C Programming
- Structures
- Functions
- Arrays
- File Handling

## How to Compile and Run

### Compile
```bash
gcc main.c -o addressbook
```

### Run
```bash
./addressbook
```
